# **App Name**: TryngkasAI

## Core Features:

- Text Summarization: Summarize text input using Gemini AI. Directly available on the landing page.
- PDF Summarization: Summarize content from uploaded PDF files using Gemini AI.  Directly available on the landing page.
- URL Summarization: Summarize content from websites via URL input using Gemini AI. Directly available on the landing page.
- Copy to Clipboard: Allow users to copy the summarized text to their clipboard.
- Download Summarized Text: Allow users to download the summarized text as a .txt file.
- File Processing Locally: Process the PDF Files Locally

## Style Guidelines:

- Primary color: Deep purple (#624CAB) for a modern and futuristic feel.
- Secondary color: Golden yellow (#FFC857) as an accent color for contrast and emphasis.
- Font: 'Nunito' (sans-serif) for both headings and body text. Note: currently only Google Fonts are supported.
- Landing page structure: Header with logo (purple text with yellow dot accent), quick links (How it Works, Features), and a yellow 'Use Now' button with purple text.  Hero section with a headline, sub-headline, and illustration of a document shrinking. Action hub with tabs (Text, PDF, URL) and a large yellow 'Summarize Now' button with loading animation.  Three cards highlighting privacy, accuracy, and instant access. Footer with 'Build with ❤️ by Trisakti Akbar (bolded)'. Additional landing page elements to extend the page downwards.
- Loading animation on the 'Summarize Now' button.
